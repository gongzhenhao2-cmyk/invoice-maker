# Invoice Maker

A local-first invoice maker for Australian contractor invoices.

## Features

- Runs in the browser with no server.
- Saves invoice data in the current browser using localStorage.
- Calculates totals from hours and hourly rate.
- Supports simple maths in hours/rate fields, such as `4.5+4.5`.
- Checks ABN checksum format.
- Checks BSB and account number format.
- Creates the next pay-period invoice from Wednesday to Tuesday.
- Optional Saturday/Sunday work selection.
- English and Chinese UI.
- Print or save as PDF from the browser.

## Privacy

This app does not send data to a server. Data is stored in the browser on the device where it is used.
